# git Domain

The `git` domain maps domain keywords to underlying faker implementations.

## Faker Documentation

- [https://fakerjs.dev/api/git](https://fakerjs.dev/api/git)

## Methods

### `git.branch`

Generates a random branch name.

- Canonical: `awd.domain.git.branch`
- Faker docs: [https://fakerjs.dev/api/git](https://fakerjs.dev/api/git)

No parameters.

Examples:

```txt
git.branch()
```

Example return values:
- `"microchip-connect"`
- `"panel-copy"`

### `git.commitDate`

Generates a date string for a git commit using the same format as `git log`.

- Canonical: `awd.domain.git.commitDate`
- Faker docs: [https://fakerjs.dev/api/git](https://fakerjs.dev/api/git)

No parameters.

Examples:

```txt
git.commitDate()
```

Example return values:
- `"Sun May 17 09:42:07 2026 -0400"`
- `"Sun May 17 20:00:50 2026 -0300"`

### `git.commitEntry`

Generates a random commit entry as printed by `git log`.

- Canonical: `awd.domain.git.commitEntry`
- Faker docs: [https://fakerjs.dev/api/git](https://fakerjs.dev/api/git)

No parameters.

Examples:

```txt
git.commitEntry()
```

Example return values:
- `"commit 3fcabfda3fe2d860f29cfb2eb4fcad5fd8d2c9b3\r\nAuthor: Macey Nolan-Lubowitz <Macey.Nolan-Lubowitz14@yahoo.com>\r\nDate: Mon May 18 01:23:37 2026 +0800\r\n\r\n    index haptic interface\r\n"`
- `"commit ca5db31dcffaec5ad95eed7fdb2eea80ffcfdae8\r\nMerge: ed0cde8 75e8be0\r\nAuthor: Lynn Koelpin <Lynn.Koelpin@gmail.com>\r\nDate: Sun May 17 12:48:57 2026 +0800\r\n\r\n    program mobile interface\r\n"`
