# book Domain

The `book` domain maps domain keywords to underlying faker implementations.

## Faker Documentation

- [https://fakerjs.dev/api/book](https://fakerjs.dev/api/book)

## Methods

### `book.author`

Returns a random author name.

- Canonical: `awd.domain.book.author`
- Faker docs: [https://fakerjs.dev/api/book](https://fakerjs.dev/api/book)

No parameters.

Examples:

Shows the default book.author call.

```txt
book.author
```

Returns: `Ian McEwan`
