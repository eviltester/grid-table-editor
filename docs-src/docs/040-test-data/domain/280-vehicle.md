# vehicle Domain

The `vehicle` domain maps domain keywords to underlying faker implementations.

## Faker Documentation

- [https://fakerjs.dev/api/vehicle](https://fakerjs.dev/api/vehicle)

## Methods

### `vehicle.bicycle`

Returns a type of bicycle.

- Canonical: `awd.domain.vehicle.bicycle`
- Faker docs: [https://fakerjs.dev/api/vehicle](https://fakerjs.dev/api/vehicle)

No parameters.

Examples:

Shows the default vehicle.bicycle call.

```txt
vehicle.bicycle
```

Returns: `Flat-Foot Comfort Bicycle`
