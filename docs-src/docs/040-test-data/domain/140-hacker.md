# hacker Domain

The `hacker` domain maps domain keywords to underlying faker implementations.

## Faker Documentation

- [https://fakerjs.dev/api/hacker](https://fakerjs.dev/api/hacker)

## Methods

### `hacker.abbreviation`

Returns a random hacker/IT abbreviation.

- Canonical: `awd.domain.hacker.abbreviation`
- Faker docs: [https://fakerjs.dev/api/hacker](https://fakerjs.dev/api/hacker)

No parameters.

Examples:

```txt
hacker.abbreviation()
```

Example return values:
- `"TLS"`
- `"SSD"`
