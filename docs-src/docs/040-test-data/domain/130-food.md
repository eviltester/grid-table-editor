# food Domain

The `food` domain maps domain keywords to underlying faker implementations.

## Faker Documentation

- [https://fakerjs.dev/api/food](https://fakerjs.dev/api/food)

## Methods

### `food.adjective`

Generates a random dish adjective.

- Canonical: `awd.domain.food.adjective`
- Faker docs: [https://fakerjs.dev/api/food](https://fakerjs.dev/api/food)

No parameters.

Examples:

Shows the default food.adjective call.

```txt
food.adjective
```

Returns: `juicy`
