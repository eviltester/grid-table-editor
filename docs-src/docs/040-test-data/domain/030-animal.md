# animal Domain

The `animal` domain maps domain keywords to underlying faker implementations.

## Faker Documentation

- [https://fakerjs.dev/api/animal](https://fakerjs.dev/api/animal)

## Methods

### `animal.bear`

Returns a random bear species.

- Canonical: `awd.domain.animal.bear`
- Faker docs: [https://fakerjs.dev/api/animal](https://fakerjs.dev/api/animal)

No parameters.

Examples:

Shows the default animal.bear call.

```txt
animal.bear
```

Returns: `Giant panda`
