# git Domain

The `git` domain maps domain keywords to underlying faker implementations.

## Faker Documentation

- [https://fakerjs.dev/api/git](https://fakerjs.dev/api/git)

## Methods

### `git.branch`

Generates a random branch name.

- Canonical: `awd.domain.git.branch`
- Faker docs: [https://fakerjs.dev/api/git](https://fakerjs.dev/api/git)

No parameters.

Examples:

```txt
git.branch()
```

Example return values:
- `array-compress`

### `git.commitDate`

Generates a date string for a git commit using the same format as `git log`.

- Canonical: `awd.domain.git.commitDate`
- Faker docs: [https://fakerjs.dev/api/git](https://fakerjs.dev/api/git)

No parameters.

Examples:

```txt
git.commitDate()
```

Example return values:
- `Tue Apr 28 04:28:58 2026 -0600`

### `git.commitEntry`

Generates a random commit entry as printed by `git log`.

- Canonical: `awd.domain.git.commitEntry`
- Faker docs: [https://fakerjs.dev/api/git](https://fakerjs.dev/api/git)

No parameters.

Examples:

```txt
git.commitEntry()
```

Example return values:
- `commit 4f9a2d1c Author: Alex Example <alex@example.com> Date: Tue May 19 2026`

### `git.commitMessage`

Generates a random commit message.

- Canonical: `awd.domain.git.commitMessage`
- Faker docs: [https://fakerjs.dev/api/git](https://fakerjs.dev/api/git)

No parameters.

Examples:

```txt
git.commitMessage()
```

Example return values:
- `reboot cross-platform system`

### `git.commitSha`

Generates a random commit sha.

- Canonical: `awd.domain.git.commitSha`
- Faker docs: [https://fakerjs.dev/api/git](https://fakerjs.dev/api/git)

No parameters.

Examples:

```txt
git.commitSha()
```

Example return values:
- `3418f0e64e8eae52ebd67b11d98e571fd6a81017`
