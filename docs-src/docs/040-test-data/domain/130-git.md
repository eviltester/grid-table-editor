# git Domain

The `git` domain maps domain keywords to underlying faker implementations.

## Faker Documentation

- [https://fakerjs.dev/api/git](https://fakerjs.dev/api/git)

## Methods

### `git.branch`

Generates a random branch name.

- Canonical: `awd.domain.git.branch`
- Faker docs: [https://fakerjs.dev/api/git](https://fakerjs.dev/api/git)

No parameters.

Examples:

```txt
git.branch()
```

Example return values:
- `"circuit-synthesize"`
- `"bus-connect"`

### `git.commitDate`

Generates a date string for a git commit using the same format as `git log`.

- Canonical: `awd.domain.git.commitDate`
- Faker docs: [https://fakerjs.dev/api/git](https://fakerjs.dev/api/git)

No parameters.

Examples:

```txt
git.commitDate()
```

Example return values:
- `"Sun May 17 21:43:47 2026 -0200"`
- `"Sun May 17 17:21:57 2026 -0500"`

### `git.commitEntry`

Generates a random commit entry as printed by `git log`.

- Canonical: `awd.domain.git.commitEntry`
- Faker docs: [https://fakerjs.dev/api/git](https://fakerjs.dev/api/git)

No parameters.

Examples:

```txt
git.commitEntry()
```

Example return values:
- `"commit 7d02da9f9e8cbf9e1e750a8efdcf55de1defa011\r\nMerge: c18a99c d5f13aa\r\nAuthor: Vada_Muller <Vada_Muller52@hotmail.com>\r\nDate: Sun May 17 17:32:17 2026 -0200\r\n\r\n    synthesize mobile system\r\n"`
- `"commit 8fec128f93dfecd026a5b1b62f269e35afaa11d7\r\nAuthor: Regan Labadie <Regan_Labadie52@yahoo.com>\r\nDate: Mon May 18 01:40:23 2026 +0600\r\n\r\n    parse auxiliary program\r\n"`
